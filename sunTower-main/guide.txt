SunTower local setup guide

1. Yeu cau moi truong
- Java 21
- MySQL
- Maven Wrapper (`mvnw`, `mvnw.cmd`) da co san trong repo

2. Cau hinh du an
- Du an doc bien moi truong tu file `.env`
- Co the tao file `.env` moi tu `.env.example`
- Profile mac dinh trong bo cau hinh hien tai la `mysql`
- Co them profile `local-nooauth` de chay local khi chua cau hinh OAuth
- Co them profile `test` dung H2 in-memory cho integration test

3. Tao co so du lieu
- Mo thu muc `estate_db_generator`
- Su dung file `sql_estate.sql` de tao schema/du lieu nen
- Neu can bo du lieu bo sung cho hoa don, xem `generate_invoice_data.py`

4. Chay ung dung
- Mo terminal tai thu muc `sunTower-main`
- Neu may chua dat `JAVA_HOME`, cau hinh truoc khi chay:
  `$env:JAVA_HOME="C:\Program Files\Java\jdk-21"`
- Chay lenh:
  `.\mvnw.cmd spring-boot:run`
- Neu muon bo qua OAuth khi chay local:
  `$env:SPRING_PROFILES_ACTIVE="mysql,local-nooauth"`
  `.\mvnw.cmd spring-boot:run`

5. Chay test
- Chay toan bo test:
  `.\mvnw.cmd test`
- Test profile hien tai dung H2 in-memory qua file `src/main/resources/application-test.properties`
- Profile `test` tu dong gom `local-nooauth`
- Tai khoan seed cho test auth API:
  - `api_admin` / `12345678`
  - `api_staff` / `12345678`
  - `api_customer` / `12345678`
- Integration test auth hien co trong:
  `src/test/java/com/estate/api/AuthV1ApiIntegrationTest.java`

6. URL chinh
- Trang public: `http://localhost:8080/suntower`
- Trang dang nhap: `http://localhost:8080/login`
- Trang admin: `http://localhost:8080/admin/dashboard`
- Trang staff: `http://localhost:8080/staff/dashboard`
- Trang customer: `http://localhost:8080/customer/home`

7. API auth moi
- `POST /api/v1/auth/login`
- `GET /api/v1/auth/me`
- `POST /api/v1/auth/logout`
- `POST /api/v1/auth/forgot-password`
- Dang nhap API se cap cookie:
  - `estate_access_token`
  - `estate_refresh_token`
